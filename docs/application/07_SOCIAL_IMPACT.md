# 07. 사회적 가치 (Social Impact)

> 예비창업패키지 양식 매핑: **가산점 — 사회적 가치 / ESG / 지속가능성**

---

## 1. 사회 문제 정의

**보행 약자 1,400만 명이 "오늘 외출할 수 있는가"를 매일 새로 결정해야 하는 사회.**

- 65세↑ 외출 포기 → **신체 위축·인지 저하·사회적 고립**
- 장애인 이동권 제약 → **고용·교육·문화 참여 기회 손실**
- 영유아 동반자 외출 어려움 → **육아 우울·고립감 증가**
- 가족(보호자) 불안 → **간병 부담·자녀 경력 단절**

> 보행 약자의 **'자립적 외출'은 사회적·경제적 비용 절감의 출발점**입니다.

---

## 2. 정량 지표 (측정 가능)

### 2.1 사용자 외출 활동 변화
| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| **월 평균 외출 횟수** | GPS 트립 기록 비교 (가입 전 자기보고 vs 가입 후 실제) | +20% (12개월) |
| **월 평균 외출 거리** | GPS 누적 거리 | +30% (12개월) |
| **외출 시간대 다양성** | 시간대별 외출 분포 | 점심·저녁 시간 진입 확대 |
| **혼자 외출 비율** | 보호자 동행 vs 단독 | +15% (12개월) |

### 2.2 안전·건강
| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| **체류 자동 감지 횟수** | useGPS.js isStaying 트리거 | 적절한 휴식 유도 |
| **SOS 발동·해소 시간** | SOS.jsx 로그 | 평균 < 3분 |
| **낙상 사고 감소** | 자치구 보건소 통계 협력 | -10% (서울 5개 구) |

### 2.3 보호자 안심
| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| **NPS (보호자)** | 분기별 설문 | > 50 |
| **간병 스트레스 감소** | 표준 척도 (ZBI 등) | -15% |
| **부모 외출 인지 시간** | 알림 수신 ~ 확인까지 | 평균 < 5분 |

### 2.4 사회 경제 효과
| 지표 | 측정 방법 | 목표 |
|------|-----------|------|
| **요양 시설 입소 지연** | 베타 사용자 추적 (3년) | 평균 6개월 지연 |
| **간병인 비용 절감** | 가족 자기 진단 설문 | 월 10만 원/가구 |

## 5. 공공 데이터 환원 계획

### 5.1 시민 → 정부 환원
- **커뮤니티 제보 (공사·통행불가·계단)** 익명 집계
- 자치구 정책 자료로 제공
- 무장애 환경 개선 우선순위 시각화

### 5.2 익명 GPS 트립 분석
- 보행 약자의 실제 이동 경로 패턴
- 보행 친화도 부족 구간 식별
- 자치구·시 정책 결정 지원 (개인정보 완전 제거)

### 5.3 오픈데이터 기여
- OSM (OpenStreetMap) 한국 무장애 정보 기여
- OSRM 한국 도보 프로파일 개선
- data.go.kr 표준 데이터 품질 피드백

---

## 6. 윤리·개인정보 보호 원칙

### 6.1 최소 수집 원칙
- 위치 데이터: 경로 안내·체류 감지에 필요한 최소량만 보관
- 음성 데이터: 클라이언트에서 처리, 서버 미전송 (Web Speech API)
- 가족 연동: 명시적 동의·언제든 해제 가능

### 6.2 익명화
- GPS 트립: 사용자 ID와 분리, 통계 집계만 보관
- 커뮤니티 제보: 위치 좌표 + 타입만, 제보자 신원 비공개

### 6.3 투명성
- 개인정보처리방침 명문화
- 데이터 보관 기간 명시 (커뮤니티 72h, 트립 12개월)
- 사용자 본인 데이터 다운로드·삭제 권한

### 6.4 가족 연동 동의
- 양방향 동의 (1차 사용자 + 보호자)
- 일방적 추적 금지
- 가족 연결 해제 즉시 위치 공유 중단
