## 4. 자금 운용 계획 (지원금 1억 예시)

> 실제 지원 금액에 따라 비율을 유지하며 조정 가능. 아래는 **1억 원 기준 예시**.

### 4.1 항목별 예산

| 항목 | 비중 | 금액 (원) | 산출 근거 |
|------|------|-----------|-----------|
| **인건비** | 60% | 60,000,000 | 백엔드 1인·디자이너 1인·매니저 1인 × 평균 6개월 |
| **외주·용역** | 15% | 15,000,000 | UI 디자인·접근성 감수·법무·세무 |
| **클라우드·API** | 10% | 10,000,000 | Vercel Pro·DB(Supabase Pro)·FCM·Kakao 유료 호출·도메인 |
| **마케팅·시연** | 10% | 10,000,000 | 복지관 시연 키트·SNS 광고·인쇄 안내물·이벤트 |
| **기타 운영** | 5% | 5,000,000 | 사무 공간·교통·예비비 |
| **합계** | 100% | **100,000,000** | |
