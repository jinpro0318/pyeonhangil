# 04. 목표 시장 분석 (Market)

> 예비창업패키지 양식 매핑: **성장 전략 — 시장 분석 / 진입 전략**

---

## 1. 시장 규모 — TAM / SAM / SOM

### 1.1 TAM (Total Addressable Market) — 약 1,400만 명
국내 보행 보조가 필요한 전체 인구.

| 세그먼트 | 인구 | 비고 |
|----------|------|------|
| 65세↑ 고령자 | 약 950만 명 | 행안부 주민등록 2024 |
| 등록 장애인 | 약 264만 명 | 보건복지부 2023, 지체장애 44% |
| 영유아 동반 보호자 | 약 230만 가구 | 통계청 출생아 추계 |
| **중복 제외 합산** | **약 1,400만 명** | TAM |

### 1.2 SAM (Serviceable Available Market) — 약 600만 명
서울 + 수도권 거주 보행 약자 (Phase 1~2 대상).

- 서울: 약 175만 명 (65↑) + 35만 명 (장애인) + 30만 가구 (영유아) ≈ **약 200만 명**
- 경기·인천: 추가 약 400만 명
- 합계: **약 600만 명**

### 1.3 SOM (Serviceable Obtainable Market) — 3년 내 MAU 10만 명

| 시점 | MAU 목표 | 근거 |
|------|----------|------|
| M+6 | 1,000명 | 서울 5개 자치구 베타 |
| M+12 | 10,000명 | 서울 전역 + 복지관 협력 |
| M+24 | 50,000명 | 수도권 확장 + B2B 첫 계약 |
| M+36 | 100,000명 | 전국 주요 도시 + 광고·라이선스 |

> 목표 침투율: SAM의 약 1.7%. 보수적 추산.

## 2. 시장 진입 단계 — 서울 MVP → 수도권 → 전국

### Phase 1: 서울 (현재 ~ M+12)
- **현재 시드 데이터** (`public/data/`)
  - `subway_elevators.geojson` — 서울 지하철 주요역 엘리베이터 20+
  - `jangsu_chairs.geojson` — 종로·중구·동작·영등포 장수의자
  - `heat_shelters_seoul.geojson` — 서울 무더위쉼터
- **타깃 자치구**: 종로·중구·동대문·서대문·동작 (고령자 비율 높은 5개)
- **확보 채널**: 복지관·노인대학·주민센터 시연

### Phase 2: 수도권 (M+12 ~ M+24)
- **확장 방법**: data.go.kr ODCloud 전체 다운로드 자동화 (현재 5,000건 캐시 → 지역별 캐시 분리)
- **타깃 도시**: 인천·수원·고양·성남·용인
- **자치구 협력 모델**: 베타 자치구가 데이터 큐레이션 환원

### Phase 3: 전국 (M+24 이후)
- **광역 확장**: 부산·대구·광주·대전·울산 + 도 단위
- **자치단체 라이선스**: B2B 라이선스 (지자체 정책 데이터 환류)
- **다언어**: 영어·중국어·일본어 관광객 무장애 안내

> Phase 1은 **현재 시드 데이터와 완전히 일치**하므로 즉시 실행 가능.

## 3. 경쟁사 분석

### 3.1 직접 경쟁 (국내)

| 서비스 | 강점 | 약점 | 편한길과 차이 |
|--------|------|------|---------------|
| **카카오맵** | POI 풍부, 국민 인지도 | 시각 중심, 자동차 우선 | 보행 약자 특화 부재 |
| **네이버지도** | POI·실내지도 | 무장애 정보 미흡 | 음성 양방향 부재 |

### 3.2 인접 경쟁 (국내)

| 서비스 | 영역 | 편한길과 차이 |
|--------|------|---------------|
| 서울시 **'장애 없는 서울'** | 정보 포털 | 정적 정보, 내비 기능 없음 |
| 행안부 **'장애 없는 생활환경'** | 인증·정책 | 일반 시민 사용 X |
| 복지관 **개별 앱** | 회원 관리 | 지도·경로 기능 없음 |

| 서비스 | 국가 | 시사점 |
|--------|------|--------|
| **Wheelmap** | 독일 | 휠체어 접근성 평점 (OSM 기반) — 커뮤니티 제보 모델 참고 |
| **AccessNow** | 캐나다 | 장애인 접근성 리뷰 — 평점 시스템 참고 |

## 4. 시장 트렌드 — 지금이 적기

### 4.1 인구 구조 (Pull)
- 2030년 65세↑ 인구 **25% 진입** → 보행 약자 시장 30%+ 성장
- 1인 고령가구 증가 → 가족이 원격으로 챙겨야 할 부모 증가 → **보호자 사용자도 확보**

### 4.2 정책 (Push)
- 서울시 **'보행친화도시 서울'** 비전 2030
- 행안부 **'무장애 생활환경 조성 5개년 계획'**
- 보건복지부 **'장애인 이동권 보장 강화'** 정책
- **교통약자 이동편의 증진법** 의무화

### 4.3 기술 (Enabler)
- 무료/저비용 지도 API 성숙 (Kakao 500만/월 무료)
- Web Speech API → 클라우드 음성 비용 ≈ 0
- data.go.kr 표준 데이터 일괄 다운로드 가능
- Vercel·Cloudflare Edge → Serverless 비용 선형 증가

### 4.4 자본·투자 (Capital)
- **사회적 임팩트 투자** 확대 (한국임팩트투자네트워크)
- **소셜벤처 정부 지원** (중기부·서울시·SBA)
- **ESG 기업 협업** 기회 (대기업 CSR)

## 5. 진입 장벽 (Barrier to Entry)

### 5.1 우리가 쌓을 진입 장벽
- **데이터 큐레이션 자산**: 서울 시드 → 자치구 환류 → 독점적 무장애 POI 데이터셋
- **커뮤니티 네트워크**: 72h 제보가 누적되면 신규 진입자가 따라잡기 어려움
- **사용자 습관**: 음성 안내·체류 감지가 익숙해지면 전환 비용 발생
- **B2B 계약**: 자치구·복지관 라이선스가 lock-in 효과

### 5.2 잠재 진입자에 대한 대응
- **Google Maps Accessibility**: 한국어 음성·국내 POI 데이터·자치단체 협력에서 우위
- **카카오맵 진입 가능성**: 카카오와 데이터 협력·라이선스 모델 가능 (적이 아닌 동맹)

## 6. 시장 검증 계획

| 단계 | 활동 | 검증 지표 |
|------|------|-----------|
| M+0~3 | 5개 자치구 베타 (100명) | DAU, 1주 잔존율 |
| M+3~6 | 복지관 시연 + 인터뷰 | NPS, 사용 의향 |
| M+6~9 | 서울 전역 1,000명 | 월 외출 횟수·거리 증가율 |
| M+9~12 | B2B 첫 계약 | 라이선스 1건 이상 |
