## 0. 이 문서 묶음에 대하여

본 폴더 `docs/application/` 는 **'모두의 창업' 챌린지 신청서 작성**만을 위한 전용 문서 모음입니다.
프로젝트 루트의 `README.md`(개발자용 기술 문서)와 **물리적으로 격리**되어 있으며, 신청서 양식이 변경되거나 방향이 전환되어도 기존 코드·문서에는 영향이 없도록 설계되었습니다.

- **양식 기준**: 예비창업패키지 표준 (문제인식 → 실현가능성 → 성장전략 → 팀구성)
- **시장 범위**: 서울 MVP → 수도권 → 전국
- **작성 원칙**:
  1. 모든 사실 주장에 **코드 인용 (`파일:라인`)** 동반 → 심사 시 검증 가능
  2. "현재 구현 vs 앞으로 만들 것"을 **시점 명확히 분리** (06_ROADMAP 중심)
  3. 사용자가 직접 채워야 할 부분은 `<!-- TODO: 사용자 입력 -->` 주석으로 표시
  4. 10개 파일이 **독립적**이라 다른 챌린지 양식으로도 재조합 가능

## 5. 파일 위치 약속

- **모든 신청서 문서**: `docs/application/` 하위에만 작성
- **루트 `README.md`**: 개발자용 기술 가이드 — **수정 금지**
- **코드 (`src/`, `api/`, `public/`)**: 신청서 작성 단계에서는 **변경 없음**
- **샘플 데이터·실명·예산 수치**: 모두 `<!-- TODO -->` 주석으로 표시 → 사용자가 최종 검토 시 채움

## 6. 챌린지 양식 변경 시 재조합 가이드

만약 다른 챌린지(서울시 청년창업·사회적기업 진흥원 등)로 변경되더라도:

- `01_OVERVIEW` `02_PROBLEM` `03_SOLUTION` — **공통 활용** (어느 양식이든 필수)
- `04_MARKET` `05_BUSINESS_MODEL` — 일반 창업 양식 활용
- `07_SOCIAL_IMPACT` — 사회적기업·사회적가치 양식 시 비중 확대
- `06_ROADMAP` `08_TEAM_BUDGET` — 지원 기간·예산 규모에 따라 수치 조정

각 파일은 독립적이므로 **필요한 섹션만 추출해 재구성 가능**합니다.
