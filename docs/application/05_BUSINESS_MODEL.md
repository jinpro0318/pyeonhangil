# 05. 사업화 전략 (Business Model)

> 예비창업패키지 양식 매핑: **성장 전략 — 사업화 / 수익 모델 / GTM**

---

## 1. 수익 모델 — 3단계 진화

### Phase 1 (M+0~12): 무료 + 정부 지원금
- **목적**: 사용자 확보·데이터 누적·신뢰 형성
- **비용 충당**: 예비창업패키지 등 정부 지원금
- **사용자 비용**: 0원
- **수익**: 없음 (사용자 확보가 KPI)

### Phase 2 (M+12~36): B2B2C
- **B2B 라이선스** (지자체·복지관·요양원)
  - 무장애 데이터 환류 + 관리자 대시보드
  - 예시 단가: 자치구당 연 1,000~3,000만 원
- **B2C 광고** (안내소·약국·병원·복지 의류)
  - 무장애 시설 카드 내 협력사 노출
  - 사용자 클릭당 30~100원
- **데이터 분석 리포트** (지자체·연구기관)
  - 익명화된 보행 패턴 정책 자료
  - 예시 단가: 리포트당 500~2,000만 원

### Phase 3 (M+36~60): 프리미엄 + API
- **B2C 프리미엄** (월 3,900원)
  - 광고 제거, 가족 다인(5인+), 고급 통계, 음성 옵션
  - 목표 전환율: MAU의 3~5%
- **API 라이선스** (3rd party 앱)
  - 무장애 POI API 월 종량제
  - 보행 약자 경로 API
- **글로벌 확장** (영·중·일)
  - 관광객 대상 무장애 안내 라이선스

## 2. 수익 추정 (5년)

| 연도 | MAU | 매출원 | 연 매출 (예상) |
|------|-----|--------|----------------|
| Y1 | 10,000 | 정부 지원금 | — (지원금 외 0) |
| Y2 | 30,000 | B2B 2건 + 광고 | 약 6,000만 원 |
| Y3 | 50,000 | B2B 5건 + 광고 + 리포트 | 약 2.5억 원 |
| Y4 | 80,000 | B2B 10건 + 프리미엄 1% + 광고 | 약 6억 원 |
| Y5 | 100,000 | B2B 20건 + 프리미엄 3% + API | 약 12억 원 |

> 보수적 추산. ARPU 약 $5/년 (월 약 500원 광고 + 프리미엄 3%).

## 3. 고객 유치 전략 (GTM)

### 3.1 1차 사용자 (보행 약자)
| 채널 | 활동 | 비용 |
|------|------|------|
| **복지관/노인대학** | 오프라인 시연 + 강사 파트너십 | 낮음 (관계 비용) |
| **주민센터** | 무장애 정책 캠페인 동참 | 낮음 |
| **자치구청 협력** | 베타 5개 구 보도자료 | 중간 |
| **종합병원 대기실** | 진료 대기 중 안내 | 중간 |

### 3.2 세컨더리 사용자 (보호자·가족)
| 채널 | 활동 | 비용 |
|------|------|------|
| **맘카페·블로그** | 유모차 코너 후기 | 낮음 |
| **카카오톡 채널** | 가족 SOS·도착 알림 사례 공유 | 중간 |
| **유튜브** | "내 부모님 외출 안심" 시리즈 | 중간 |
| **언론 PR** | 사회적 임팩트 스토리 | 중간 |

### 3.3 B2B 채널
| 채널 | 활동 | 비용 |
|------|------|------|
| **지자체 정책 제안** | 보행친화도시·무장애 조성 사업 입찰 | 높음 |
| **복지관 협회** | 공동 컨퍼런스 | 중간 |
| **요양원·실버타운** | 가족 안심 솔루션 패키지 | 높음 |

## 6. 핵심 가설과 검증 계획

| 가설 | 검증 방법 | 성공 기준 |
|------|-----------|-----------|
| **H1**. 보행 약자는 음성 안내를 선호한다 | 베타 100명 NPS 조사 | NPS > 30 |
| **H2**. 보호자는 가족 SOS에 가치를 느낀다 | 보호자 설문·인터뷰 | 80%+ '가치 있음' |
| **H3**. 자치구는 무장애 데이터 환류에 예산을 쓴다 | 5개 구 협의 | 1건 이상 B2B 계약 |
| **H4**. 사용 후 외출 빈도가 증가한다 | 가입 전후 GPS 트립 비교 | 외출 +20% |
