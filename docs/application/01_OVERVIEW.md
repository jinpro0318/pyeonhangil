## 2. 30초 엘리베이터 피치

> "기존 지도 앱은 자동차·시각 중심입니다. **편한길**은 고령자·장애인·유아동반자가 화면을 보지 않고도 버스·지하철과 걷는 구간을 한 흐름으로 음성 안내받고, 무장애 시설(쉼터·화장실·엘리베이터)을 자동으로 알려줍니다.
> 3분 이상 멈추면 자동으로 쉼터 모드로 전환되고, 가족에게 위치가 자동 공유됩니다.
> 서울 5개 자치구 MVP → 수도권 → 전국 확장으로, 약 **1,400만 명**의 보행 약자가 자립적으로 외출하는 사회를 만듭니다."

## 3. 핵심 가치 3가지

### ① 🎙 Voice-First — 화면을 보지 않아도 이동
- **STT(음성 인식) + TTS(음성 합성)** 양방향 (`src/hooks/useVoice.js`, `src/hooks/useSpeechRecognition.js`)
- 검색·경로 안내·도착 확인 모두 음성으로 처리 (`src/pages/Search.jsx`, `src/pages/Navigation.jsx`)
- 시각 부담을 줄여 고령자·시각장애인의 진입장벽 해소

### ② 🪑 무장애 경로 + POI 통합
- 도보 전용 경로 (Tmap → OSRM foot → Kakao → 직선) 4단 폴백 (`api/route.js:40-124`)
- **3계층 POI 통합**: 정적 GeoJSON(서울 시드) + Kakao Local + data.go.kr 공공데이터 (`src/services/poiApi.js:138-184`)
- 경로 주변 50m 반경 자동 필터링 → 쉼터·화장실·엘리베이터·경사로 즉시 표시 (`src/pages/RouteSuggest.jsx:150-164`)

### ③ 👨‍👩‍👧 가족 자동 연동
- 3분·10m 반경 자동 체류 감지 → 쉼터 모드 자동 전환 (`src/hooks/useGPS.js:70-92`)
- SOS 3단계: 119 전화 / 가족 위치 전송 / 가장 가까운 쉼터 안내 (`src/pages/SOS.jsx`)
- 보호자(가족)는 실시간 위치·상태 모니터링

## 4. 현재 단계 (2026-05 기준)

### MVP 완성 항목 (`src/App.jsx:1-60`, 14개 페이지)

```
온보딩       Splash → Intro → Permissions → WalkState
메인         Home (음성 검색·즐겨찾기·빠른 찾기)
검색         Search (음성/텍스트, 자동완성)
경로         RouteSuggest (이동수단 스위처·50m 필터·커뮤니티 제보 표시)
지도         MapMain (반경 1500m POI)
이동         Navigation (실시간 음성·체류 감지)
상태         Resting (자동 쉼터) · Arrived (도착)
긴급         SOS (3단계 도움)
커뮤니티     Community (6종 제보·72h 유효)
설정         Family · MyInfo
```

### 검증 가능한 기술 자산
- **코드**: 14 페이지 · 5 hooks · 4 API 서비스 · 88줄 유틸 (geo.js)
- **데이터**: 서울 지하철 엘리베이터·장수의자·무더위쉼터 GeoJSON 시드
- **API**: 4종 Vercel Serverless Functions (route / local / govdata / place)
- **배포**: Vercel + Kakao Maps 도메인 화이트리스트 운영
